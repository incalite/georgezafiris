Projects

  • ECPR (E-commerce Product Entity) This is an experimental client for managing products of an ecommerce entity. It communicates with a custom API built from scratch with plain PHP 7 \w PDO and MySQL. The client was built using Vanilla JavaScript, AJAX and Bootstrap 5.
  • Questioning System A fully functioning dashboard for in-house-used questioning system made using Laravel, Bootstrap, jQuery, Popper.js, SaSS and Vue.
  • Ticket Tracker The TT is a professional ticket tracking dashboard, that lets companies manage details about their clients, developers and tasking tickets. TicketTracker also uses sophisticated algorithms to monetize each ticket according the corresponding client and ticket description.
  • Vacation Corporate Manager Based on a fictional concept, employees can log in and request vacation days. A supervisor/administrator will log in as well and approve or reject requests sent by each user.
  • DodgeCube This is a very simple game written in C++ & SDL2. You are a red cube that needs to dodge other cubes. Once the score reaches 100 points the game is won. I made this game to get a better handle regarding some the core concepts of SDL2, using Surfaces, Textures, Audio, Events and game state management.
  • Case Studies Python is one of the most popular general use languages out there. It's easy to learn if you're a beginner, but it gets harder once you understand its overall logic. This thread's a storage place for some of my python case studies.
  • The Nosco System Nosco's a tiny clone of StackOverflow aimed to students and their curriculum. Feel free to clone and dive in. Made using the LAMP stack.
  • NPM Module - Population Factory Population Factory generates an entire population to test your API, Web Application, you name it. Each individual citizen has the conventional characteristics of the everyday life (tax, name, lastname, address, etc) and some additional medical parameters such as Step Count, Calories, HR(Heart Rate) etc.
